Built-in Tools

All tools are sandboxed to the project directory. Defaults: read-only is allowed, mutating tools require permission. The auto-edit badge in the chat header bypasses the permission prompt for every tool while it's on.

Ignore Patterns

Paths matching these patterns are skipped by file-listing/search tools and refused by file-touching tools. Goal: keep small models from wasting context on caches, deps, and build outputs — not to filter file types. Patterns use .gitignore syntax.

Honor project .gitignore
