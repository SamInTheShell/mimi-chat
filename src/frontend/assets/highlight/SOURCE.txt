Theme:
  hljs-theme.css is hand-written for Mimi against project CSS variables
  (theme.css). It is NOT a stock highlight.js theme — no upstream theme is
  vendored. The class names (.hljs, .hljs-keyword, .hljs-string, …) match
  highlight.js' canonical token taxonomy so any future stock theme would also
  work as a drop-in.
